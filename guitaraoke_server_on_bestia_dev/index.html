Guitaraoke Leader for Android

Play guitar and sing with your friends

The lyrics and chords show in sync with the "Leader" on any smartphone, tablet, computer or smart TV.


Installation, first run and settings

1. Only the Leader needs to install this android app. The Followers will use their existing internet browser.

2. The "Leader" installs the last APK for android from the Github release page.

Download the last APK for android

3. This app is safe (no special permission required on android),
Temporarily change your settings to install this APK: "Settings - Security - Unknown Sources".
Read more here. After installation revert the setting back.

4. Enable the hotspot on your smartphone (share your internet connection, it will create a local wi-fi network).

5. Open GuitaraokeLeader app for the first time. For security reasons you have to manually create your Guitaraoke folder.
Go to the folder Music and create a new sub-folder `Guitaraoke`.
Inside your Guitaraoke folder you can create sub-folders for different type of music: romantic, rock, traditional,...
Finally "Allow" the app to access this folder.

6. Click on "Download song", choose the songs to download. They will be downloaded in the background.

7. Click on the app title "Guitaraoke Leader" in the left upper corner to go back to the first page.

8. You can also copy any mp4 file using android file manager to your Guitaraoke folders. That is much faster.
Create your own Guitaraoke song collections.

9. Create a QRcode image for your wifi connection. Some smartphone have this functionality integrated or go to qrty.
Store the image in your GuitaraokeLeader folders as `wifi_qrcode.jpg` size 200x200px.

10. Practice and rehearse with GuitaraokeLeader before going public ! Try also the QRCodes before public use.


Go live - instructions for the Leader

1. Limit the internet traffic, change the "Mobile networks - Network mode" to "2G only". After using GuitaraokeLeader, return the "Network mode" to LTE.

2. Enable the hotspot on your smartphone (share your internet connection, it will create a local wi-fi network).

3. Open GuitaraokeLeader app, choose your Guitaraoke folder and allow access.

4. Click on QRCode. It will show 2 QRCodes. First is for the wifi connection, second to open the local web page.
All modern smartphone can use this 2 QRCodes. Let the followers read them with their smartphones.
When their application is running, they can share the same QRCodes to other people.
Older phones have to do it old-school: connect to your wifi with a password and write manually the address in the browser 192.168....:8080 (it is in the upper right corner).

5. The Leader can see the number of Followers. Ask if everybody is connected!

6. The Leader clicks on a song, then clicks on the Play button and finally on the FullScreen button.

7. Start to play-a-long and sing-a-long.

8. You can use the back button to exit FullScreen.

9. You can Stop the song and return to the song list.


Super short instructions for the Followers

1. The follower reads the first QRCode on the Leader phone for the wifi connection.
Older phones have to do it old-school, choose the network and enter the password.

2. The follower reads the second QRCode on the Leader phone. It will open the browser on the local web page.
Older phones must type manually the address in the browser from the right upper corner of the Guitaraoke Leader.

3. The followers can share these QRCodes with other followers, so the Leader is not overwhelmed.

5. The Follower clicks on the button "Full screen" and enjoys the play-a-long and sing-a-long.
